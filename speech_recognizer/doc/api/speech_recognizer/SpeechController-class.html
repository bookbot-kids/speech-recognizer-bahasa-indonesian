flushSpeech({String toRead = '', String grammar = ''}) → Future<void>
when flushSpeech is called, a string is passed containing the text of what we expect to hear from the user next The toRead is distinct from grammar because the former should be used as the audio clip recording transcript, whereas the latter needs to be for the speech recognition model. For example, if we expect the word "cat", then toRead is set to "cat" but grammar needs to be set to cat, mat, sat etc. Otherwise, the recognizer will only ever return the word "cat", no matter what was said.
